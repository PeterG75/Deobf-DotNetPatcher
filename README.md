an basic deobfuscator for last version (at the moment) dotnetpatcher (v4.5.9.0) https://bitbucket.org/3DotDev/dotnet-patcher

* ~ MathFixer
* ~ Sizeof Fixer
* ~ String Decryption
* ~ Integer Decryption
* ~ Control Flow Deobfuscation
* ~ Antis (AntiDump, AntiDebug, AntiTamper)
* ~ ProxyFixer
